# Security Policy

## Reporting Security Issues

If you discover a security vulnerability in this fork (RD-Agent-with-Claudex), please report it via [GitHub Issues](https://github.com/rozwer/RD-Agent-with-Claudex/issues) with the label `security`.

For vulnerabilities in the upstream Microsoft RD-Agent project, please report them to the [Microsoft Security Response Center (MSRC)](https://msrc.microsoft.com/create-report).

## Scope

This policy applies only to code and configuration added or modified in this fork. The upstream RD-Agent codebase is maintained by Microsoft under their own [security policy](https://github.com/microsoft/RD-Agent/blob/main/SECURITY.md).
